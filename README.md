    ruby 2.6.8
    bundler 2.3.13
    minitest 5.15.0

    ruby main.rb

        bundler
        minitest

        bundle exec ruby spec/grid_spec.rb
        bundle exec ruby spec/robot_spec.rb
        bundle exec ruby spec/helper_spec.rb
